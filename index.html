Backup & Data Export
One-Click Backup
Downloads all your files, conditions, notes, settings, and custom fields as a single JSON file. Store it in iCloud Drive, an external drive, or anywhere safe. If Supabase ever went away, this file contains everything needed to rebuild your entire database on any platform.
Where to store your backup file
1. After clicking Download Backup, a file named zaphire-backup-[date].json will appear in your Downloads folder.
2. Move it somewhere safe. Recommended: iCloud Drive → Zaphire Capital → Backups
3. Do this once a month, or before making any major changes.
How to restore or migrate to another platform
If you ever need to move away from Supabase, your JSON backup file can be imported into any of the following:

Supabase (new project): Go to supabase.com → SQL Editor and use the Table Editor to import the JSON directly.

Firebase / Firestore: Use the Firebase Console to import the JSON collections.

PostgreSQL / any SQL database: The JSON maps directly to four tables: files, conditions, settings, and customFields.

Local SQLite: Use DB Browser for SQLite (free app) → Import JSON → create matching tables.

Need help migrating? Open a new conversation with Claude, say "I have a Zaphire Capital backup JSON and need to migrate to [platform]" and attach the file. Claude will write the exact import steps.
Important
Your backup file contains sensitive client information. Treat it like any confidential business document. Do not share it via email or store it in a public location. iCloud Drive with a strong Apple ID password is a safe choice.
